Banking System (Java OOP Project)

Description

This project is a simple Banking System developed using Java and Object-Oriented Programming concepts. It simulates basic banking operations.

Features

* Account creation
* Deposit money
* Withdraw money
* Check account balance

Concepts Used

* Encapsulation
* Inheritance
* Abstraction
* Object-Oriented Programming (OOP)

Technologies

* Java

Purpose

This project was created to understand how real-world banking systems work using OOP principles.
